7. Python Script to find the largest and smallest elements in an array.


```
arr = [10, 5, 20, 8, 15]

largest = arr[0]
smallest = arr[0]

for num in arr:
    if num > largest:
        largest = num
    if num < smallest:
        smallest = num

print("Largest element:", largest)
print("Smallest element:", smallest)
```
