

```

1. From the Navigation, select Kubernetes Engine.
2. Click Create Cluster.
3. Switch to the standard cluster.
4. Name the cluster boutique-demo.
5. Select the Zonal option and pick the zone closest to your location.
6. Click the default-pool and enter 5 in the Number of nodes text.
7. Click the Nodes.
8. Select e2-medium (2 vCPU, 4 GB memory) from the Machine Configuration Machine type drop-down.
9. Click Create to create the cluster
```
